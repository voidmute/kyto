# Wiki source

Markdown here is the **source of truth** for the [GitHub Wiki](https://github.com/voidmute/kyto/wiki).

**Browse now (no wiki tab required):** [docs/wiki/Home.md](Home.md) in this folder on GitHub.

## One-time GitHub Wiki setup

GitHub creates the `kyto.wiki` git repository only after the first wiki page exists. Do this **once**:

1. Open https://github.com/voidmute/kyto/wiki
2. Click **Create the first page**
3. Title: `Home` — body: `Loading…` (any placeholder)
4. Save

Then run **Actions → Sync Wiki → Run workflow**, or push any edit under `docs/wiki/`.

The workflow publishes all pages and replaces the placeholder.

## Editing

Edit files in `docs/wiki/` on `main`. Do **not** edit the GitHub Wiki UI directly — changes will be overwritten on the next sync.
