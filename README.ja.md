## クイックスタート

```bash
git clone https://github.com/voidmute/kyto.git && cd kyto
./asm/build.sh && ./bin/kura-asm install
kura compile
```

## `.kyto.config`

`DOMAIN`、`USERS`、環境変数を定義。`+` でコメント。
